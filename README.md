# 🏠 HomeGPT - Your Personal AI Assistant

Welcome to **HomeGPT** – a beautifully crafted, multi-feature personal assistant built using **Python** and **Streamlit**. This project aims to simplify your digital life while offering a stunning interactive experience.

---

## 🌟 Features

* 🔐 **User Authentication**
  Secure login/register system with:

  * Login count tracking
  * Last login timestamp
  * Logout functionality

* 📁 **Memory & Password Vaults**
  Store your thoughts and passwords securely using **SQLite** backend.

* 🎵 **Music Assistant**
  Type your favorite song, and it opens instantly as a YouTube video in the app.

* 🧠 **ChatGPT Integration**
  Talk to your AI assistant and get real-time responses in an intuitive chat window.

* 📝 **Quiz Mode**
  Engage with topic-based quizzes for fun and learning.

* 📨 **Telegram Messaging**
  Send real-time messages to your personal Telegram using the Telegram Bot API.

* 🎈 **Delightful UX/UI**

  * Lottie animations (balloons, home entrance)
  * Time-based greetings (Good Morning, Evening, etc.)
  * Soothing login sound & visual confetti
  * Beautiful sidebars with user stats

## ✨ How to Run Locally

```bash
# Clone the repository
https://github.com/YOUR_USERNAME/homegpt

# Navigate to project folder
cd homegpt

# Install dependencies
pip install -r requirements.txt

# Run the Streamlit app
streamlit run app.py
```

Make sure to:

* Add your Telegram Bot credentials in `.env`
* Place the required `.wav` files in the `/assets` folder
